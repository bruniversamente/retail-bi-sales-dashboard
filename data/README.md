# Dados do projeto

Os arquivos nesta pasta são dados sintéticos criados para simular uma operação de varejo multicanal. Nenhum dado é real ou sensível.

## Arquivos

- `sample_orders.csv`: pedidos, datas, canal, status, estado e forma de pagamento.
- `sample_order_items.csv`: itens dos pedidos, preço, custo, desconto e quantidade.
- `sample_products.csv`: cadastro de produtos, categorias e subcategorias.
- `sample_customers.csv`: cadastro sintético de clientes e segmentos.
- `sample_targets.csv`: metas mensais por canal.

## Uso

Os CSVs de exemplo são pequenos para facilitar leitura e revisão no GitHub. O script `scripts/generate_retail_data.py` gera uma base maior para análises e dashboards.
